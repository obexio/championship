IP — доставка пакетов по сети.
TCP — надёжная передача, всё по порядку.
UDP — быстрая передача без гарантий.
DNS — переводит имя сайта в IP.
HTTP — запросы к сайтам (без шифрования).
HTTPS — HTTP, но зашифрованный.
FTP — передача файлов.
SFTP — безопасная передача файлов.
SSH — безопасное удалённое подключение к серверу.
SMTP — отправка почты.
IMAP — получение почты, хранение на сервере.
POP3 — получение почты, скачивает на устройство.
DHCP — выдаёт IP-адреса устройствам.
ARP — сопоставляет IP → MAC адрес.
ICMP — пинговка, проверка доступности.
SSL/TLS — шифрование между клиентом и сервером.
NTP — синхронизация времени.
SNMP — управление сетевым оборудованием.
RDP — удалённый рабочий стол Windows.
Telnet — старый удалённый доступ без шифрования.
LDAP — управление пользователями/группами в домене.
Kerberos — авторизация в Windows-доменах.
SIP — сигналы для звонков / VoIP.
WebSocket — двухсторонняя связь в реальном времени.
MQTT — маленькие сообщения для IoT устройств.
BGP — протокол, по которому интернет знает куда отправлять трафик.
SMB — сетевой обмен файлами (Windows).
TFTP — простой, небезопасный FTP без авторизации.
mDNS — локальное обнаружение устройств (типа AirDrop).
VLAN (802.1Q) — логическое разделение сети.
STP — защита от петель в сети.
